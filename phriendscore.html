PhriendScore

Vue / Vuex / Vuetify / Netlify/ Node / Express / JWT / MongoDB Atlas / Heroku / Responsive Design

PhriendScore is my passion project. It's an idea for an app I had for ages that started as a spreadsheet for tracking all of the Phish concerts I've seen. Once COVID hit I took the time quarantining as an opportunity to make the dream a reality. Not only did I end up learning Vue, Vuex for state management, and Vuetify, but I really grew my JS/MongoDB chops by utilizing virtual database properties to take a single html string from the Phish.net API and parsing it into personalized concert statistics thats hundreds of 'Phans' enjoy.
