IrelandUpdate

HTML / CSS / Materialize / JavaScript / jQuery / Node / NPM / Express / Handlebars / APIs / MongoDB / Responsive Design

Inspired by my Irish immigrant parents, I decided to make an app so that they can have a one-stop shop for all of the newest headlines from their homeland. The full-stack app collects articles from their favorite news outlets, saves them to a MongoDB. The app uses Handlebars for templating via the Express server. Axios GET requests are then made to the specified sites, cheerio is used to scrape the article data, and a newly created object is inserted into the MongoDB collection.The articles are then retrieved from the DB, and jQuery is used to populate the collapsing table
