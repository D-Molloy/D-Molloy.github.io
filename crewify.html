Crewify

HTML / CSS / Bootstrap / JavaScript / React / Node / NPM / Express / MongoDB / Auth0 / AJAX / APIs / Responsive Design

With technology being ever-present in our daily lives, it was surprising to learn how prevalent paper-based work orders are in today's businesses. Crew-based companies send their crews out for the day with a physical list of jobs to do and more often than not that list is lost before lunch. Efficiency plummets, appointment windows are missed, customers are unhappy, and management has no reliable way to track job details and completion. That's why we built Crewify; to bring efficiency and accountability to crew-based companies. Crewify is a full-stack crew management platform built using the MERN stack (MongoDB, Express, React, and Node). As the Lead Frontend Developer, I designed and implemented portals for both management and crews. Managers can create crews and jobs, as well as search jobs to review status and completion data, while a mobile-first interface was designed for crews, so that they can quickly access dynamically created navigation and telephone links on their mobile device, view customer info, enter job notes/completion time, and mark jobs as complete.
