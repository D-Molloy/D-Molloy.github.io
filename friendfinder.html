FriendFinder

HTML / CSS / Materialize / JavaScript / jQuery / Node / NPM / Express / AJAX / AJAX / APIs / Responsive Design

I was the sole developer of FriendFinder, a responsive full-stack app. When the user completes the survey, the form is validated, an AJAX request sent to an API sends back data from other users, and (via an algorithm I created), the current user's score is compared to each of the saved user's data, and the user with the closest absolute score to the current user is shown as their new best friend. I had a lot of fun creating this site and showcasing my creative muscle in user experience, copywriting, and developing code that works. Give it a try!
