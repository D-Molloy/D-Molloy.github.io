Weather Redux

HTML / CSS / Bootstrap 4 / JavaScript / React / Redux / NPM / OpenWeatherMap API / Google Maps API

I'm a data-driven person and I've always loved data presented in a visually compelling way. That's why I had such a great time building this React app while also learning Redux. When you search for a city, the fetchWeather action makes an api call to openweathermap.org API via axios and then passes the request promise to the reducer. The redux-promise npm package in the reducer pulls the weather data from the promise and updates the state with the new data. This info is then passed to the WeatherList container and the necessary location data is extracted in order to display the map for each city (via the Google Maps API in the google_map.js component) along with the upcoming weather data that is passed down to the chart.js component (charts are generated using the Sparklines npm package). Feel free to clone the repo and try it for yourself!
